completion-progress
===================

DSL for defining objects completion level (score and percentage), like LinkedIn or Facebook, but works with every object!

## Requirements

* Ruby 1.9+

## Installation

Add it to your Gemfile:

`gem 'completion-progress'`

Then:

`bundle`
